##### 懒加载：

先只加载页面上所看到的图片，等滚动到页面下面时，再加载所需的图片，可以缓解浏览器压力，增强用户体验

##### 瀑布流布局：

宽度一致，而高度不一样的布局

##### 实现原理：

1. 用ajax获取 page = i 的 n 条数据
2. 把数据拼装成dom放在页面上
3. 使用瀑布流摆放dom位置
4. 当页面滚动到底部（某个条件），page++ ，循环1
   - 在最底部设置一个隐藏的透明元素，当该元素出现在视窗里，再次发送请求
